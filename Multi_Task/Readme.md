在自动驾驶等领域大多采用多任务学习，如特斯拉等视觉方案厂商。
1. 多任务之间可相互辅助，提高模型表现，高效完成环境感知等任务；
2. 多任务之间可充当正则项，提高模型泛化性能；
3. 并行完成多项任务相较单独完成多项任务，提高模型推理速度，有利于硬件部署与工业应用生产。
